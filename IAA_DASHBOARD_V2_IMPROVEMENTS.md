## مقاييس IAA المدعومة

### Cohen's Kappa
- **الاستخدام**: قياس الاتفاق بين موسِّمين اثنين
- **الصيغة**: κ = (P_o - P_e) / (1 - P_e)
  - P_o: نسبة الاتفاق المرصود
  - P_e: نسبة الاتفاق المتوقع بالصدفة

### Fleiss' Kappa
- **الاستخدام**: قياس الاتفاق بين عدة موسِّمين
- **الصيغة**: κ = (P̄ - P̄_e) / (1 - P̄_e)
  - P̄: متوسط نسبة الاتفاق المرصود
  - P̄_e: متوسط نسبة الاتفاق المتوقع

## مقياس التفسير

| النطاق | التفسير | الرمز |
|--------|---------|------|
| < 0 | أقل من الصدفة | ⚠️ |
| 0.0 – 0.2 | ضعيف جداً | △ |
| 0.2 – 0.4 | ضعيف | ○ |
| 0.4 – 0.6 | معتدل | → |
| 0.6 – 0.8 | جيد | → |
| 0.8 – 1.0 | ممتاز | ✓ |

## المكونات الرئيسية

### KappaBadgeV2
عرض بصري محسّن لقيمة Kappa مع تفسير فوري.

```tsx
<KappaBadgeV2 kappa={0.85} label="Cohen's Kappa" />
```

### KappaBarV2
شريط تقدم ديناميكي يعرض قيمة Kappa مع تدرج لوني.

```tsx
<KappaBarV2 kappa={0.75} maxHeight={300} />
```
