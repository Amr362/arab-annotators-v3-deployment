تم تطوير وتحسين صفحة **Inter-Annotator Agreement (IAA) Dashboard** بشكل جذري لتوفير تجربة مستخدم أفضل وتصورات بيانات أكثر وضوحاً وفعالية.

## 🎯 التحسينات الرئيسية

### 1. **واجهة رسومية محسّنة (UI/UX)**

#### قبل التحسين:
- تصميم بسيط وأساسي
- عرض محدود للمعلومات
- عدم وجود تنظيم واضح للمحتوى

#### بعد التحسين:
- ✅ تصميم حديث وجذاب يتوافق مع السمة السعودية
- ✅ بطاقات ملخصة (Summary Cards) توضح المقاييس الرئيسية بشكل فوري
- ✅ تنسيق احترافي مع ألوان متناسقة وسهلة القراءة
- ✅ استخدام الرموز (Icons) لتحسين الفهم البصري

### 2. **رسوم بيانية تفاعلية (Interactive Charts)**

تم إضافة عدة أنواع من الرسوم البيانية:

#### أ. **Bar Chart** — مقارنة مقاييس Kappa
```
- يعرض مقارنة بصرية بين Cohen's Kappa و Fleiss' Kappa
- ألوان ديناميكية بناءً على قيمة Kappa
- تفاعل فوري عند تمرير الماوس (Hover)
```

#### ب. **Progress Bars** — نسبة الاتفاق
```
- عرض بصري لنسبة الاتفاق بين المرمزين
- شريط تقدم متدرج اللون
- نسبة مئوية واضحة بجانب الشريط
```

#### ج. **Kappa Progress Bar** — عرض متقدم لقيمة Kappa
```
- خط صفر في المنتصف يفصل بين القيم الموجبة والسالبة
- ألوان ديناميكية تعكس جودة الاتفاق
- نسبة مئوية معايرة من -1 إلى 1
```

### 3. **نظام التبويبات (Tabbed Interface)**

تم تنظيم المحتوى في 4 تبويبات رئيسية:

#### 📊 **تبويب النظرة العامة (Overview)**
- رسم بياني مقارنة Kappa
- عرض نسب الاتفاق
- ملخص سريع للمقاييس

#### 🔄 **تبويب المقارنة (Comparison)**
- عرض جنباً إلى جنب لـ Cohen's و Fleiss' Kappa
- معلومات تفصيلية لكل مقياس
- تفسيرات واضحة للنتائج

#### 📈 **تبويب التفاصيل (Details)**
- معلومات شاملة عن كل مقياس
- جداول توضيحية للبيانات الإحصائية
- شرح مفصل للتفسيرات

#### 📏 **تبويب المقياس (Scale)**
- مقياس Landis & Koch الكامل
- شرح لكل نطاق قيم
- ملاحظات توضيحية مهمة

### 4. **بطاقات المقاييس الرئيسية (Summary Cards)**

تم إضافة 4 بطاقات رئيسية توضح:

| البطاقة | المحتوى |
|--------|--------|
| **Cohen's Kappa** | قيمة Kappa للمقارنة الثنائية + التفسير |
| **Fleiss' Kappa** | قيمة Kappa للمقارنة المتعددة + عدد الموسِّمين |
| **الاتفاق المرصود** | نسبة الاتفاق الفعلي بين المرمزين |
| **جودة البيانات** | تقييم شامل لجودة البيانات المتاحة |

## 📊 المقاييس المدعومة

### Cohen's Kappa (ثنائي):
```
- يقيس الاتفاق بين موسِّمَين فقط
- يصحح الاتفاق بالصدفة
- الصيغة: κ = (Po - Pe) / (1 - Pe)
  حيث:
    Po = الاتفاق المرصود
    Pe = الاتفاق المتوقع بالصدفة
```

### Fleiss' Kappa (متعدد):
```
- يقيس الاتفاق بين أكثر من موسِّمَين
- يعمل عندما يكون عدد الموسِّمين مختلفاً لكل مهمة
- تعميم لـ Cohen's Kappa
```

### مقياس Landis & Koch:
```
< 0.0     : أقل من الصدفة
0.0–0.2   : ضعيف جداً
0.2–0.4   : ضعيف
0.4–0.6   : معتدل
0.6–0.8   : جيد
0.8–1.0   : ممتاز
```

## 🎓 التعليمات التعليمية

### فهم Cohen's Kappa:
```
مثال:
- الموسِّم 1: [A, B, A, B, A]
- الموسِّم 2: [A, B, A, B, B]
- الاتفاق المرصود: 4/5 = 80%
- Cohen's Kappa: يحسب الاتفاق بعد تصحيح الصدفة
```

### فهم Fleiss' Kappa:
```
مثال مع 3 موسِّمين:
- المهمة 1: [A, A, B] → اتفاق جزئي
- المهمة 2: [A, A, A] → اتفاق تام
- Fleiss' Kappa: يحسب الاتفاق الإجمالي
```
