# دليل الإعداد على Railway

## المتغيرات البيئية المطلوبة

```
DATABASE_URL=postgresql://...      # رابط قاعدة البيانات (إلزامي)
JWT_SECRET=your-secret-key-here    # مفتاح الجلسة (إلزامي - أي نص عشوائي)
```

## متغيرات اختيارية

```
# إنشاء حساب المدير الأول تلقائياً عند التشغيل
ADMIN_EMAIL=admin@example.com
ADMIN_PASSWORD=your-strong-password
ADMIN_NAME=Admin

# Google OAuth (اختياري - فقط إن أردت تسجيل الدخول بجوجل)
GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
VITE_GOOGLE_CLIENT_ID=...          # نفس القيمة - مطلوبة في build
```

## طريقة تسجيل الدخول

### الطريقة الأولى: إعداد يدوي (الأسهل)
1. افتح الموقع لأول مرة
2. ستجد صفحة "إعداد أولي"
3. أدخل بيانات المدير وأنشئ الحساب
4. ستدخل تلقائياً

### الطريقة الثانية: من env vars (للـ CI/CD)
ضع `ADMIN_EMAIL` و `ADMIN_PASSWORD` في Railway
الحساب يُنشأ تلقائياً عند أول تشغيل

## إنشاء حسابات المصنفين
من لوحة Admin → المستخدمون → إنشاء جماعي
يمكن إنشاء 50 حساب دفعة واحدة مع تحميل كلمات المرور كـ CSV

## تسجيل دخول المصنف
- اسم المستخدم: annotator_01 (النمط الذي تختاره عند الإنشاء)
- كلمة المرور: كما في الـ CSV الذي نزّله المدير
